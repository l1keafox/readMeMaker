# Readme Maker

  [![License: ISC](https://img.shields.io/badge/License-ISC-blue.svg)](https://opensource.org/licenses/ISC)

  Using Node.js this creates a readme.md for future repos.

  ## Usage 
  Run 'node index.js' and answer the prompts given

  1. Download repo 
2. Run "node index.js"
